Diy Benchtop Power Supply

A benchtop power supply is useful for anyone wanting to start a hobby in electronics. This power supply was made with a soldering iron and a few parts off amazon. This would be a great beginning project and adds a tool that you can use for future projects. To build the enclosure I used a 3D printer however you can use either some acrylic or some wood and some hand tools to build a encloser for the electronics
